🎵 Custom Music Player

A modern, customizable music player that delivers a premium experience — completely free.

This project allows users to enjoy a smooth and stylish music streaming interface, while also giving them the ability to request their favorite songs, which can be added to the platform.

🚀 Features
🎧 Clean and modern UI
🎶 Smooth audio controls (Play, Pause, Next, Previous)
📱 Responsive design (Mobile + Desktop)
💎 Premium feel without any cost
📝 Song request feature (Users can request songs to be added)
⚡ Fast and lightweight performance

💡 How It Works

Users can browse and play available songs.
If a song is not available, they can request it.
The requested song can then be reviewed and added to the player.
Once added, it becomes available for everyone.

🛠️ Tech Stack

HTML5
CSS3
JavaScript
Web3Forms

The goal of this project is to create a free music player with a premium feel, focusing on:
